[В прошлом примере](https://github.com/extrafix/python/blob/main/3/game.py)
 реализовал ранее проработанные АТД и методы работы с ними.

Дополнил более дополнительным логированием очков здоровья после хода.


Класс Battle имеет 3 метода, которые оперируют с иницилизированными в одном из них сущностями бойцов.   
1. start_battle запускает битву: опрашивает бойцов о выборе части тела для атаки/защиты в рамках их хода.  
2. calculate_damage вычисляет урон с учетом защиты.
3. is_continue применяется для определения критерия для окончания схватки (сейчас выполняется только 1 ход).  


Класс Hod позволяет считать со консольнго ввода и сохранить выбор части для атаки/защиты с обрабткой ошибок ввода.  
1. make_hod - Выполнение хода игроком.    
2. try_make_hod - Выполнение хода игроком с обработкой ошибки неверного аргумента.  


Класс Warrior хранит инфрмацию о силе и здоровье бойца.  
1. hit - Получение урона бойцом.
